## ⚡ 性能优化建议

### 如果人很多变得卡慢：

#### 1. 优化数据库
```sql
-- 添加索引
CREATE INDEX idx_task_column ON sb_task(column_id);
CREATE INDEX idx_task_board ON sb_task(board_id);
```

#### 2. 增加内存
- 确保至少 8GB RAM
- 推荐 16GB

#### 3. 调整MySQL配置
```ini
max_connections = 200
innodb_buffer_pool_size = 1G
```

#### 4. 使用CDN
- 静态资源（HTML/CSS/JS）放CDN
- 减少服务器压力
