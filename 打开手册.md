# 📖 SyncBoard 打开手册

## 🚀 快速开始（3步启动）

### 第1步：启动服务
双击运行：`启动服务.bat`

等待看到：`🚀 SyncBoard 启动成功！`

---

### 第2步：访问页面
浏览器打开：http://localhost:8080/api/login.html

---

### 第3步：注册登录
1. 点击"立即注��"
2. 填写用户名、邮箱、密码
3. 注册后登录使用

---

## 📱 详细操作指南

### 一、本地使用（你自己用）

#### 1. 启动服务
```bash
双击：启动服务.bat
```

#### 2. 访问地址
- 登录：http://localhost:8080/api/login.html
- 注册：http://localhost:8080/api/register.html

#### 3. 停止服务
```bash
双击：停止服务.bat
```

---

### 二、局域网使用（同WiFi的人用）

#### 1. 查看你的IP地址
```bash
双击：查看访问地址.bat
```

#### 2. 告诉朋友访问
```
http://你的IP:8080/api/register.html
```

#### 3. 配置防火墙（首次）
```bash
双击：配置防火墙.bat
```

---

### 三、外网使用（不同地方的人用）

#### 1. 启动内网穿透
```bash
双击：C:\natapp\natapp.exe
```

#### 2. 查看外网地址
在Natapp窗口找到：`http://xxxx.natappfree.cc`

#### 3. 分享给朋友
```
http://xxxx.natappfree.cc/api/register.html
```

## 💡 基本操作

### 创建任务
1. 点击右上角 "+ 新建任务"
2. 填写任务信息
3. 点击"创建"

### 移动任务
1. 鼠标拖拽任务卡片
2. 拖到其他列
3. 松开鼠标完成移动

### 删除任务
1. 点击任务卡片右下角 🗑️ 图标
2. 确认删除

### 退出登录
1. 点击右上角用户信息旁的"退出"
2. 确认退出

## ⚠️ 重要提示

### 启动顺序（必看！）

**本地使用**：
```
启动服务.bat → 打开浏览器访问
```

**外网使用**：
```
启动服务.bat → 启动Natapp → 分享地址
```

### 必须保持运行的窗口

- ✅ SyncBoard服务窗口（不能关闭）
- ✅ Natapp窗口（外网访问时不能关闭）

### 停止服务

**本地使用**：
```bash
双击：停止服务-简单版.bat
```

**外网使用**：
1. 关闭Natapp窗口
2. 双击 `停止服务-简单版.bat`

## 🔧 常见问题

### Q1: 启动失败，提示端口被占用
```bash
双击：停止服务.bat
然后再双击：启动服务.bat
```

### Q2: 打开显示404
```
确保访问：http://localhost:8080/api/login.html
注意后面的 /api
```

### Q3: 朋友无法访问
```
1. 检查SyncBoard服务是否运行
2. 检查防火墙是否配置（双击"配置防火墙.bat"）
3. 检查是否在同一WiFi（局域网）
```

### Q4: 外网访问不了
```
1. 检查Natapp是否运行
2. 检查SyncBoard服务是否运行
3. 查看Natapp窗口是否显示 Online
```

### Q5: 在线人数显示0
```
这是正常的！当你登录后会自动计数
刷新页面后显示"1人在线"
```

## 📊 功能说明

### 看板列
- **待办事项**：还没开始的任务
- **进行中**：正在做的任务
- **已完成**：做好的任务

### 任务优先级
- 🔴 **高**：红色，紧急重要
- 🟠 **中**：橙色，一般重要
- 🟢 **低**：绿色，可以暂缓

### 在线人数
- 实时显示当前使用人数
- 登录自动计数
- 退出自动减少
