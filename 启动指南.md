# 🚀 SyncBoard 启动指南

## 📝 快速启动（三种方式）

### 方式 1: 双击启动脚本（推荐）⭐

1. **启动服务**：双击 `启动服务.bat`
2. **打开浏览器**：访问 http://localhost:8080/api
3. **停止服务**：双击 `停止服务.bat`

---

### 方式 2: 使用 PowerShell 命令

**启动**：
```powershell
cd C:\projects\SyncBoard
& "C:\Users\赵慧楠\Desktop\宁波实训\apache-maven-3.9.9-bin\apache-maven-3.9.9\bin\mvn.cmd" spring-boot:run
```

**停止**：在终端按 `Ctrl + C`

---

### 方式 3: 手动检查所有服务

1. 确认 MySQL 服务运行（任务管理器）
2. 确认 Redis 服务运行（任务管理器）
3. 执行方式 2 的启动命令

---

## ⚠️ 常见问题

### 问题 1: 双击启动脚本后立即关闭

**原因**：MySQL 或 Redis 服务未启动

**解决**：
1. 双击 `检查状态.bat` 查看具体哪个服务未启动
2. 在"任务管理器"中启动对应服务

### 问题 2: 启动脚本报"找不到 mvn.cmd"

**原因**：Maven 路径不正确

**解决**：编辑 `启动服务.bat`，修改第 19 行的 Maven 路径为你实际的 Maven 安装路径

### 问题 3: 端口 8080 被占用

**解决**：
1. 双击 `停止服务.bat` 停止旧服务
2. 重新双击 `启动服务.bat`

### 问题 4: 浏览器显示 404

**原因**：访问了错误的地址

**解决**：确保访问 http://localhost:8080/api （注意后面的 /api）
