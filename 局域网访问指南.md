# 🌐 SyncBoard 局域网访问指南

## 📱 让其他人访问你的 SyncBoard

### 方案1：同 WiFi/局域网访问（推荐）⭐

**适用场景**：你的朋友/同事连接到同一个 WiFi 或路由器

---

## 🚀 快速设置步骤

### 第1步：确认服务正在运行

确保你的 SyncBoard 服务已经启动：
- 双击 `启动��务.bat`
- 看到 "SyncBoard 启动成功！" 消息

### 第2步：获取你的局域网IP地址

**你的局域网IP地址是**：`192.168.10.22`

或者重新获取：
1. 按 `Win + R`
2. 输入 `cmd` 回车
3. 输入 `ipconfig` 回车
4. 找到 "无线局域网适配器 WLAN" 或 "以太网适配器"
5. 复制 "IPv4 地址" 后面的数字

### 第3步：测试局域网访问

在你的电脑浏览器访问：
```
http://192.168.10.22:8080/api/login.html
```

如果能打开登录页面，说明服务正常运行！

### 第4步：告诉其他人访问地址

**告诉朋友/同事访问**：
```
http://192.168.10.22:8080/api/register.html
```

他们可以：
1. 打开浏览器
2. 输入上面的地址
3. 注册账号
4. 登录使用

## ⚠️ 常见问题

### Q1: 其他人访问显示"无法连接"？

**解决方法**：

1. **检查防火墙**（Windows）：
   - 按 `Win + R`，输入 `wf.msc` 回车
   - 点击"入站规则"
   - 点击"新建规则"
   - 选择"端口" → 下一步
   - 选择"TCP"，特定本地端口输入 `8080` → 下一步
   - 选择"允许连接" → 下一步
   - 全选（域、专用、公用）→ 下一步
   - 名称输入 "SyncBoard" → 完成

2. **关闭公用网络防火墙**（简单但不安全）：
   - 控制面板 → Windows Defender 防火墙
   - 启用或关闭 Windows Defender 防火墙
   - 专用网络和公用网络都关闭

### Q2: 服务需要一直开着吗？

**是的**！其他人访问时，你的电脑需要：
1. 保持开机状态
2. 保持 SyncBoard 服务运行
3. 不要休眠或睡眠（设置电源选项为"从不"）

### Q3: 局域网访问速度慢吗？

**不慢**！局域网访问速度很快，通常在 100Mbps-1000Mbps

### Q4: 多少人可以同时使用？

**理论无限**！取决于：
- 你的电脑性能
- MySQL 和 Redis 的连接数限制
- 默认配置支持 50-100 人同时在线

## 🌍 方案2：互联网访问（外网访问）

如果想让不同地方的人访问，需要：

### 选项A：内网穿透（免费）

**推荐工具**：
1. **花生壳** - https://hsk.oray.com/
2. **Natapp** - https://natapp.cn/
3. **Frp** - https://github.com/fatedier/frp（需要服务器）

**使用步骤**（以花生壳为例）：
1. 注册花生壳账号
2. 下载花生壳客户端
3. 添加内网映射：
   - 映射类型：HTTP
   - 内网主机：192.168.10.22
   - 内网端口：8080
   - 域名：使用免费域名或购买
4. 启动映射
5. 其他人访问提供的域名地址

**优点**：
- ✅ 免费（有限制）
- ✅ 简单易用
- ✅ 不需要云服务器

**缺点**：
- ❌ 免费版有流量限制
- ❌ 速度可能较慢
- ❌ 需要保持运行

### 选项B：部署到云服务器（专业）

**推荐平台**：
1. **阿里云** - 学生机 9.9元/月
2. **腾讯云** - 轻量服务器 50元/年
3. **华为云** - 新用户优惠

**配置要求**：
- CPU：1核或2核
- 内存：2GB
- 带宽：1Mbps（可以升级）
- 系统：Ubuntu 20.04 或 CentOS 7+
- 价格：50-200元/年

**部署步骤**：
1. 购买云服务器
2. 安装 JDK 17
3. 安装 MySQL 8.0
4. 安装 Redis
5. 上传 SyncBoard 项目
6. 配置防火墙开放 8080 端口
7. 启动服务

**优点**：
- ✅ 24小时运行
- ✅ 稳定可靠
- ✅ 速度快
- ✅ 支持公网访问

**缺点**：
- ❌ 需要花钱购买
- ❌ 需要一定运维知识

### 选项C：免费云平台（有额度限制）

**推荐平台**：
1. **Render** - https://render.com/
2. **Railway** - https://railway.app/
3. **Vercel** - 主要是前端

**注意**：这些平台主要支持Node.js、Python等，Java支持较少

## 💡 提示

1. **IP地址可能会变**：重启路由器或重新连接WiFi后，IP地址可能变化，需要重新查询
2. **推荐静态IP**：在路由器设置中为你的电脑设置静态IP，避免IP变化
3. **保持服务运行**：其他人使用时，不要关闭你的电脑或SyncBoard服务
4. **定期备份数据**：定期导出MySQL数据库，防止数据丢失
