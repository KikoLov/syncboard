# 更新日志

本文档记录 SyncBoard 的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-01-31

### 新增

#### 用户功能
- ✨ 用户注册功能，支持用户名、邮箱、密码
- ✨ 用户登录功能，返回 JWT Token
- ✨ 用户��息获取和更新
- ✨ 头像显示（使用 DiceBear API）
- ✨ 会话管理和自动登录

#### 看板功能
- ✨ 创建看板功能
- ✨ 编辑看板名称和描述
- ✨ 删除看板功能
- ✨ 看板成员管理
- ✨ 公开/私有看板设置
- ✨ 看板背景颜色设置

#### 任务列功能
- ✨ 创建任务列（待办、进行中、已完成）
- ✨ 编辑列名称和颜色
- ✨ 删除列功能
- ✨ 列排序功能

#### 任务功能
- ✨ 创建任务卡片
- ✨ 编辑任务标题、描述
- ✨ 设置任务优先级（高/中/低）
- ✨ 分配任务给成员
- ✨ 任务拖拽移动（使用 Fractional Indexing 算法）
- ✨ 删除任务功能
- ✨ 任务标签管理
- ✨ 任务截止日期设置
- ✨ 任务完成标记

#### 实时协作
- ✨ WebSocket 实时通信
- ✨ 任务移动实时同步
- ✨ 任务更新实时推送
- ✨ 在线用户显示
- ✨ 在线人数统计
- ✨ 用户进入/离开通知
- ✨ Redis 发布订阅支持多实例

#### 安全功能
- ✨ BCrypt 密码加密
- ✨ JWT Token 认证
- ✨ 乐观锁并发控制
- ✨ SQL 注入防护（MyBatis-Plus）
- ✨ XSS 防护

#### 数据存储
- ✨ MySQL 数据持久化
- ✨ Redis 缓存支持
- ✨ 操作日志记录
- ✨ 数据库自动填充（创建时间、更新时间）

#### 前端功能
- ✨ 响应式设计，支持移动端
- ✨ 现代化 UI（渐变背景、流畅动画）
- ✨ 拖拽交互（HTML5 Drag and Drop API）
- ✨ 实时数据更新
- ✨ 错误提示和通知
- ✨ 加载状态显示

### 技术栈

- ✨ Spring Boot 3.1.8
- ✨ Spring WebSocket
- ✨ MyBatis-Plus 3.5.7
- ✨ MySQL 8.0
- ✨ Redis 6.x
- ✨ Druid 连接池
- ✨ FastJSON2

### 优化

- ⚡ 使用 Fractional Indexing 优化拖拽性能
- ⚡ 乐观锁避免并发冲突
- ⚡ Redis Pub/Sub 支持分布式部署
- ⚡ 数据库连接池优化
