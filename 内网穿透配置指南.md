## 🚀 方案1：Natapp（强烈推荐）⭐

### 优点
- ✅ 完全免费
- ✅ 无需安装客户端（可选）
- ✅ 稳定可靠
- ✅ 支持自定义二级域名（付费）
- ✅ 国内服务，速度快

### 使用步骤

#### 第1步：注册账号
1. 访问：https://natapp.cn/
2. 点击右上角"注册"
3. 使用手机号或邮箱注册
4. 完成实名认证（免费用户也需要）

#### 第2步：创建隧道
1. 登录后，点击"购买隧道"（免费）
2. 填写配置：
   ```
   隧道名称：syncboard
   隧道协议：HTTP
   本地端口：8080
   前置域名：免费域名（系统分配）
   ```
3. 点击"购买"（免费0元）
4. 复制获得的authtoken

#### 第3步：下载客户端
1. 访问：https://natapp.cn/#download
2. 下载Windows版本（natapp_windows_amd64.zip）
3. 解压到任意目录，例如：`C:\natapp\`

#### 第4步：启动Natapp
有两种方式：

**方式A：使用命令行**
```bash
cd C:\natapp
natapp -authtoken=你的authtoken
```

**方式B：使用配置文件（推荐）**
1. 在natapp目录创建文件 `config.ini`：
```ini
[default]
authtoken=你的authtoken
clienttoken=
log=none
loglevel=ERROR
```

2. 创建启动脚本 `启动natapp.bat`：
```batch
@echo off
cd C:\natapp
natapp
pause
```

3. 双击运行 `启动natapp.bat`

#### 第5步：获取外网地址
启动后会显示：
```
Tunnel Status Online                online_url=http://xxx.natapp.cn
version                             3.0.2
...
Forwarding                          http://xxx.natapp.cn -> http://127.0.0.1:8080
```

复制 `http://xxx.natapp.cn` 这个地址就是你的外网访问地址！

#### 第6步：告诉朋友访问
```
注册页面：http://xxx.natapp.cn/api/register.html
登录页面：http://xxx.natapp.cn/api/login.html
主应用页面：http://xxx.natapp.cn/api/index.html
```

## 🥜 方案2：花生壳（老牌工具）

### 优点
- ✅ 品牌知名度高
- ✅ 界面友好
- ✅ 自带客户端

### 缺点
- ❌ 免费版限速（1kbps）
- ❌ 需要实名认证
- ❌ 限制较多

### 使用步骤

#### 第1步：下载安装
1. 访问：https://hsk.oray.com/
2. 下载"花生壳客户端"Windows版
3. 安装并启动

#### 第2步：注册登录
1. 运行花生壳客户端
2. 注册/登录账号

#### 第3步：添加映射
1. 点击"添加映射"
2. 选择"免费域名"
3. 应用类型：HTTP
4. 内网主机：192.168.10.22（或你的局域网IP）
5. 内网端口：8080
6. 点击"保存"

#### 第4步：获取外网地址
在列表中会显示：
```
域名：xxx.oray.net
类型：HTTP
外网端口：80
```

访问地址：`http://xxx.oray.net/api/login.html`

## 🌐 方案3：Ngrok（国外工具）

### 优点
- ✅ 全球知名
- ✅ 无需注册（免费版）
- ✅ 简单易用

### 缺点
- ❌ 服务器在国外，速度慢
- ❌ 可能在国内被墙
- ❌ 免费版有限制

### 使用步骤

#### 第1步：下载
1. 访问：https://ngrok.com/
2. 注册并下载Windows版本

#### 第2步：解压使用
```bash
# 解压到C:\ngrok\

# 启动ngrok
cd C:\ngrok
ngrok http 8080
```

#### 第3步：获取地址
会显示：
```
Forwarding  https://xxxx.ngrok.io -> http://localhost:8080
```

访问地址：`https://xxxx.ngrok.io/api/login.html`

## ⚠️ 注意事项

### 安全提醒：
1. **不要暴露重要信息**：这只是测试环境
2. **定期更换密码**：保护账号安全
3. **注意流量限制**：免费版通常有流量限制
4. **不要用于生产**：免费服务不稳定

### 使用限制：
- Natapp免费版：2条隧道，可能会掉线
- 花生壳免费版：限速1kbps
- Ngrok免费版：限流

### 稳定性：
- 免费服务可能会随时中断
- 适合测试，不适合长期使用
- 如需稳定服务，建议购买云服务器
